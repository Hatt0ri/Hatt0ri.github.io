Project of engineering thesis

Perlin noise-based algorithms for procedural generation of terrain and arrangement of objects in Unity

In summary, this project is the result of actions taken by the author to design, implement and test a set of tools. These tools enable game developers to automatically generate terrain and arrange trees and additional objects on it using the Unity game engine. The user, thanks to the options available, has the ability to manipulate Perlin noise on the basis of which the skeleton of the terrain is created. The set of tools has the function of dividing the terrain into types based on its height, which allows game developers for each terrain type to adapt the color of the terrain and the way of placing additional objects. The tool kit has a module that continuously generates terrain, taking into account the movement of the observer on the stage in Unity. As a result, from the perspective of the observer generated terrain gives the impression of an infinitely large. The more distant parts of the terrain generate with less detail, which improves Unity's performance.
